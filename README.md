# Checkmarx Expert Project

## 项目简介
这个项目实现了一个基于 Python 的安全检查专家，能够识别 C 语言代码中的潜在漏洞并提供修复建议。

## 功能
- 识别代码中的安全漏洞
- 提供修复建议
- 自动修复某些漏洞
- 生成审计报告
- 运行集成测试
- 推荐最佳实践

## 如何运行
1. 确保 Python 已安装。
2. 安装依赖：
    ```bash
    pip install -r requirements.txt
    ```
3. 运行主程序：
    ```bash
    python main.py
    ```

## 目录结构
- `src/`: 源代码
- `tests/`: 测试文件
- `docs/`: 文档
